# SnakeGame
A simple snakegame for C language programming course
# 贪吃蛇游戏（Snake Game）

## 项目简介
本项目是一个基于 C/C++ 语言开发的贪吃蛇小游戏，使用 EasyX 图形库实现图形界面。玩家通过控制贪吃蛇移动，吃到食物后得分增加，随着游戏进行不断挑战更高分数。

## 开发环境
- 操作系统：Windows 11
- 开发工具：Visual Studio Code
- 编译器：MinGW g++
- 图形库：EasyX

## 功能模块
- 蛇的移动控制（W、A、S、D）
- 食物随机生成
- 分数统计
- 三种游戏难度
- 暂停与继续功能
- 历史最高分记录
- 背景音乐和游戏音效
- 游戏结束提示

## 项目结构
```
main.cpp        // 程序入口
Game.cpp/.h     // 游戏主逻辑
Snake.cpp/.h    // 蛇的实现
Food.cpp/.h     // 食物功能
Map.cpp/.h      // 地图绘制
Menu.cpp/.h     // 游戏菜单
Sound.cpp/.h    // 音效播放
sound/          // 音乐和音效资源
```

## 运行方法
1. 安装 MinGW 编译环境。
2. 配置 EasyX 图形库。
3. 使用 VS Code 打开项目。
4. 编译并运行 `main.cpp`。

## 项目特点
- 图形化界面
- 操作简单
- 支持不同难度
- 带背景音乐和音效
- 适合作为 C/C++ 程序设计课程项目
